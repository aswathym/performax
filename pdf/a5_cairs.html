Specific CAIRS algorithms are designed to support each of the six (6) assessment wizards, as depicted in the following:

Tool CAIRS Recommends
Challenge Mapping Wizard (CMW) CAIRS-CMW Relevant organizational assessments related to key organizational challenges
Assessment Design Wizard (ADW) CAIRS-ADW Relevant assessment questions, based on industry, size, and specific organizational challenges
Assessor Selection Wizard (ASW) CAIRS-ASW Optimal assessor groups, based on assessments selected
Assessment Feedback Wizard (AFW) CAIRS-AFW Relevant Ideas For Action (IFAs) and Key Performance Indicators (KPIs) to assessors
Assessment Report Wizard (ARW) CAIRS-ARW Optimal report generation parameters to the user
Action Planning Wizard (APW) CAIRS-APW Uses ANFIS algorithm to Recommend high priority actions for selection, to the expert panel members.
