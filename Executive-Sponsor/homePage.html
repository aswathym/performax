Watch a Short (2:00 minute) Video on how to conduct a 360 assessment with Performax360.

  • FOCUS

    Select assessment based on key organizational challenges
    Challenge Mapper
    • Task ID 1.1 : Enter organizational challenges in free-form text entry.
    • Task ID 1.2 : View & Select the assessment(s) recommended by the Challenge Mapper.
    • Task ID 1.3 : Purchase Assessment

  • CUSTOMIZE

    Configure assessment administration, and select expert panels to Customize the assessment questionnaire template and nominate assessors
    Assessment Configurator
    • Task ID 2.1 : Select Assessment Administrator.
    • Task ID 2.2 : Select Assessment completion timeline
    • Task ID 2.3 : Decide any reqards for assessors participation (gamification)
    • Task ID 2.4 : Decide whether the assessment will be anonymous
    • Task ID 2.5 : Decide assessment result sharing permissions.
    Assessment Designer
    • Task ID 2.6 : View 100 Critical Success Factors (CSFs) for each assessment
    • Task ID 2.7 : Select relevant assessment questions by CSF context: (1) People & Leadership, (2) Analytics & Insights, (3) Strategy & Planning, (4) Execution & Process, and (5) Performance & Results.
    • Task ID 2.8 : Collaborate and Finalize CSFs for the survey questionnaire
    • Task ID 2.9 : Update the questionnaire 'CSF list' on an ongoing basis
    • Task ID 2.10 : Approve the proposed assessment questionnaire.
    Assessment Selector
    • Task ID 2.11 : View assessor groups recommended by CAIRS
    • Task ID 2.12 : Collaborate with other expert panel members
    • Task ID 2.13 : Nominate the assessor groups and assessors collaboratively
    • Task ID 2.14 : Update the 'assessor list' on an ongoing basis
    • Task ID 2.15 : Approve the proposed assessor list
    Email Editor
    • Task ID 2.16 : View & Edit 'invitation email' templates for different user groups
    • Task ID 2.17 : Approve the Invitation Email Copy.

  • ENGAGE

    Start the 360 degree assessment feedback process, and monitor progress
    Feedback Manager
    • Task ID 3.1 : Provide accurate feedback on CSF effectiveness ratings.
    • Task ID 3.2 : Submit accurate text input describing reasons for CSF reatings.
    • Task ID 3.3 : Collaborate with Assessors in the 'collaboration hub'
    • Task ID 3.4 : Submit ongoing changes to assessors ratings for CSF's
    • Task ID 3.5 : Monitor assessors feedback submission progress
    • Task ID 3.6 : Send 'reminder emails' to assessors

  • EVALUATE

    Review assessment results, identification of strengths and weeknesses, and action recommendation
    Result Reporter
    • Task ID 4.1 : View online real-time assessment results dashboard
    • Task ID 4.2 : Download PDF copy of assessment results reports.
    • Task ID 4.3 : Download Powerpoint Presentations copy of assessment results
    • Task ID 4.4 : Share assessment results with 'PerformaxShare'

  • ACT

    Select an expert panel to recommend an action plan; and appoint Key Persons Accountable(KPAs) to implement selected action items
    Accountability Coordinator
    • Task ID 5.1 : Review Ideas for Action (IFA) prioritized by ANFIS
    • Task ID 5.2 : Collaborate with other Expert Panel members
    • Task ID 5.3 : Submit recommended list of IFAs for approval
    • Task ID 5.4 : Approve selected IFAs for implementation
    • Task ID 5.5 : Nominate Key Persons Accountable (KPAs) for IFA implementation
    • Task ID 5.6 : Submit ongoing feedback and new IFAs for approval

  • MONITOR

    Monitor progress on action plan implementation, and review ongoing improvements in effectiveness ratings
    Results Tracker
    • Task ID 6.1 : Review assessment feedback results, related to the selected IFA and CSF
    • Task ID 6.2 : Form a team by nominating Key Persons Responsible (KPRs), Key Persons to be Informed (KPTI), and Key Persons to be Consulted (KPC)
    • Task ID 6.3 : Identify and Assign required tasks to KPRs
    • Task ID 6.4 : Implement tasks and actions, to improve CSF effectiveness ratings
    • Task ID 6.5 : Support the KPA and KPRs to successfully implementation tasks
    • Task ID 6.6 : Update the Performax360 system on task implementation progress
    • Task ID 6.7 : Update Assessors on actions taken to implement the IFA, related to the CSF
    • Task ID 6.8 : Revise and Update feedback and CSF effectiveness ratings
    • Task ID 6.9 : Review progress on CSF effectiveness ratings over time
